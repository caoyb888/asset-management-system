# 系统管理模块 — 任务分解及开发计划

> **项目**：产城公司资产管理系统  
> **模块**：系统管理（Module 7）  
> **技术栈**：Spring Boot 3.x + MyBatis-Plus + Redis + Vue 3 + Element Plus  
> **文档版本**：v1.0 · 2026-02-15

---

## 一、模块概览

系统管理模块是整个资产管理平台的基础支撑模块，负责用户身份认证、组织权限管控、基础数据字典及业务算法配置。本模块的稳健性直接影响所有其他业务模块的正常运行。

**数据库表清单（共18张）**

| 分类 | 数据表 |
|------|--------|
| 用户与组织 | sys_user、sys_org、sys_org_project、sys_post |
| 角色与权限 | sys_role、sys_user_role、sys_menu、sys_role_menu、sys_role_data |
| 业务字典 | sys_dict_type、sys_dict_data、sys_category、sys_code_rule |
| 租费算法 | sys_rent_algorithm、sys_rent_algo_step |
| 系统运维 | sys_config、sys_oper_log、sys_login_log |

**API前缀**：`/api/v1/system`（认证接口 `/api/v1/auth`）

### TASK-SYS-01 | 认证中心（Auth）

**技术要点**：JWT双令牌（Access Token 30min + Refresh Token 7天存Redis），登录失败锁定，密码BCrypt加密

| 编号 | 任务 | 说明 |
|------|------|------|
| 01-01 | 实现登录接口 `POST /api/v1/auth/login` | 用户名/密码校验，返回双Token；记录 sys_login_log |
| 01-02 | 实现Token刷新接口 `POST /api/v1/auth/refresh` | Redis分布式锁防并发刷新 |
| 01-03 | 实现登出接口 `POST /api/v1/auth/logout` | 删除Redis中Token，记录登出日志 |
| 01-04 | 实现JWT过滤器（JwtAuthFilter） | 解析Token，加载用户权限至SecurityContext |
| 01-05 | 登录失败锁定机制 | 连续失败N次后锁定账号（lock_time），阈值可配置 |
| 01-06 | 获取当前用户信息接口 `GET /api/v1/auth/info` | 返回用户基础信息 + 菜单权限列表 |
| 01-07 | 获取用户动态路由接口 `GET /api/v1/auth/user-routes` | 前端动态路由使用 |
| 01-08 | 前端登录页面（LoginPage.vue） | 账号/密码表单，记住密码，错误提示 |
| 01-09 | 前端路由守卫（router.beforeEach） | Token校验，动态路由注册，权限兜底404 |
| 01-10 | Pinia用户Store（useUserStore） | 保存Token、用户信息、权限标识列表 |

| 优先级 | 技术难点 | 攻关时机 | 关键方案 |

## 六、依赖与风险

| 风险点 | 等级 | 应对措施 |
|--------|------|---------|
| 租费算法公式引擎选型（Aviator vs SpEL） | 中 | 提前做POC验证，优先选Aviator（性能更好） |
