|--------|------|------|
| P0 | 商铺拆分对话框 | 选择商铺 → 填写子铺位信息 → 面积自动校验 |
| P0 | 商铺合并对话框 | 多选商铺 → 填写合并后信息 → 面积汇总展示 |
| P1 | Excel 导入对话框 | 文件上传 + 错误回显 |

#### 业务规则备注
```
拆分：
  - 原商铺标记 is_deleted=1
  - 创建多个新商铺
  - biz_shop_relation 写入 relation_type=1（拆分）
  - 校验：SUM(新商铺面积) == 原商铺面积

合并：
  - 被合并商铺均标记 is_deleted=1
  - 创建一个新商铺
  - biz_shop_relation 写入 relation_type=2（合并）
  - 校验：新商铺面积 == SUM(被合并商铺面积)
```
