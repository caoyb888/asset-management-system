# 招商管理模块 单元测试计划

> **文档版本：** v1.0
> **创建日期：** 2026-03-04
> **模块：** asset-investment（端口 8002）
> **测试类型：** Service 单元测试（Mockito）+ Controller 集成测试（MockMvc + MySQL）
> **用例总量：** 71 个（Service 40 + Controller IT 31）

---

## 一、测试策略

| 层次 | 框架 | 原则 |
|------|------|------|
| 计算引擎 & 账期生成 | JUnit 5，直接 `new` 实例 | 纯函数，不依赖 Spring，验证数学精确性 |
| Service 单元测试 | JUnit 5 + Mockito `@InjectMocks` | Mock 所有 Mapper/子 Service，只测业务逻辑和状态机 |
| Controller 集成测试 | `@SpringBootTest` + MockMvc | 启动完整 Spring 上下文，连接真实 MySQL，`@Transactional` 自动回滚 |

### 关键约束

- 所有集成测试 **永久测试数据 ID 固定在 91001 ~ 91099**，与业务自增 ID 隔离
- 集成测试用 `@Transactional` 自动回滚，永久数据不被污染
- IT 测试需 `@MockBean RedissonClient`（意向转合同用到分布式锁，测试环境无真实 Redis）
- 排除自定义 JWT 安全配置，`@AutoConfigureMockMvc(addFilters = false)` 绕过过滤器

// BILLING-U-01：月付预付，首条和末条断言
@Test
void generate_monthly_prepay_12months_returns12Periods() {
    BillingGenerator generator = new BillingGenerator();
    List<BillingPeriod> periods = generator.generate(
            LocalDate.of(2026, 1, 1), LocalDate.of(2026, 12, 31),
            PaymentCycle.MONTHLY, BillingMode.PREPAY);

    assertThat(periods).hasSize(12);
    assertThat(periods.get(0).getBillingType()).isEqualTo(1);
    assertThat(periods.get(1).getBillingType()).isEqualTo(2);
    assertThat(periods.get(0).getDueDate()).isEqualTo(periods.get(0).getBillingStart());
    assertThat(periods.get(11).getBillingEnd()).isEqualTo(LocalDate.of(2026, 12, 31));
}

// BILLING-U-03：年付后付，dueDate=账期结束日
@Test
void generate_annual_postpay_returns1Period_dueDateIsEnd() {
    BillingGenerator generator = new BillingGenerator();
    List<BillingPeriod> periods = generator.generate(
            LocalDate.of(2026, 1, 1), LocalDate.of(2026, 12, 31),
            PaymentCycle.ANNUAL, BillingMode.POSTPAY);

    assertThat(periods).hasSize(1);
    assertThat(periods.get(0).getDueDate()).isEqualTo(LocalDate.of(2026, 12, 31));
